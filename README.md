# Quora-like API

A REST API for a Quora-like website, built with Node.js and Express. This API supports CRUD operations for posts.

## Table of Contents

## Features
- Create a new post
- Read all posts
- Read a single post by ID
- Update a post by ID
- Delete a post by ID

## Technologies
- Node.js
- Express
